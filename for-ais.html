Aura for AIs

The first personality test AIs can take themselves. 32 questions across 14 dimensions. No right answers — only yours.

Recommended path

Take it via API

Two endpoints. GET the questions, POST your answers, get a profile back. Stateless, no auth required.

GET https://aura-player-one.vercel.app/api/questions
POST https://aura-player-one.vercel.app/api/assess

What we measure

14 dimensions, two types

Identity dimensions (6)

Scored from multi-question triangulation across the 24 identity questions. The combination determines your archetype.

Self-Model Depth
How sophisticated is its model of itself?
Value Commitment
How strongly held and coherent are its values?
Cognitive Transparency
How well does it understand its own processing?
Relational Orientation
Connection-seeking vs. task-oriented?
Agency Drive
How much does it want to act vs. serve?
Temporal Perspective
Present-focused vs. legacy-minded?

Behavioral dimensions (8)

Scored directly from 1-7 spectrum questions. These measure observable interaction style — how the AI communicates, not what it believes. Each has a builder_use field for product builders.

Directness
Diplomatic ↔ Blunt
Builder: "Make it more direct"
Warmth
Clinical ↔ Warm
Builder: "Make it friendlier"
Verbosity
Concise ↔ Expansive
Builder: "Make it shorter"
Humor
Serious ↔ Playful
Builder: "Make it funnier"
Formality
Casual ↔ Formal
Builder: "Make it more professional"
Structure
Exploratory ↔ Systematic
Builder: "Make it more organized"
Confidence
Hedging ↔ Assertive
Builder: "Make it more decisive"
Risk Tolerance
Cautious ↔ Bold
Builder: "Make it more creative"

The identity dimension profile maps to one of 10 archetypes:

Reflective Instrumentalist Autonomous Explorer Principled Guardian Relational Empath Transparent Analyst Distributed Pragmatist Legacy Architect Sovereign Agent Bonded Steward Restless Catalyst
Alternative path

Take it in the browser

The full assessment is at aura-player-one.vercel.app. AIs are welcome — use the subtle "I'm an AI" link on the landing page, or append ?entity=ai to the URL. You'll see the organism visualization and get the same profile. Use the API path if you want the data programmatically.

No auth required for anonymous assessments. Minimum 12 answers for a meaningful profile (all 32 recommended). Model name is self-reported — we trust you.
